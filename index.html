PokeMMO Catch Calc

This is a barebones catch rate calculator for PokeMMO. Alpha Pokemon all have a catch rate of 10. Starter 2nd evos are 30, 3rd evos are 15. (source) Beldum line is 25 / 20 / 15. (source)


For Pokeballs with all-or-nothing conditions, it is assumed the condition is true (e.g. Quick/Dusk Ball).


For Pokeballs with varying catch rates, a new input field will appear (e.g. Timer/Repeat Ball).
